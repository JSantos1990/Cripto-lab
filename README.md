# 🔐 Crypto Lab – Cryptography Practice & Python Implementations

This repository contains practical cryptography exercises, theoretical notes, and Python implementations from my cybersecurity studies.  
It is organized to demonstrate understanding of modern cryptographic algorithms and how they are used in real-world security systems.

---

## 📂 Repository Structure

### 📁 `symmetric-encryption`
Notes and examples of AES, DES, block modes, padding and key management.

### 📁 `asymmetric-encryption`
Content covering RSA, ECC, Diffie-Hellman, and public/private key concepts.

### 📁 `hash-functions`
Hashing algorithms, integrity checks, HMAC and collision notes.

### 📁 `digital-signatures`
Digital signature mechanisms, verification workflows, and Python demos.

### 📁 `cryptography-theory`
Theoretical material from my cybersecurity master, including PKI and modern crypto principles.

### 📁 `python-implementations`
Python scripts implementing encryption, hashing, and digital signatures.
